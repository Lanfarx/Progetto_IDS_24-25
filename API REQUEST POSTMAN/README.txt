Per testare le API di questo progetto con Postman:

1. Scarica Postman al seguente link (https://www.postman.com/downloads/).
2. Scarica il file della collection in questa stessa cartella.
3. Apri Postman e vai su **File > Import**.
4. Seleziona il file `.json` scaricato.
5. Ora puoi eseguire le richieste API direttamente da Postman per testare il nostro progetto.
